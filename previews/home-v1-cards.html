AXIOM
Crypto Arbitrage Screener
AXIOM — это платформа для мониторинга арбитражных возможностей на криптовалютном рынке. Мы агрегируем данные с 8 крупнейших бирж в реальном времени и показываем расхождения цен, которые можно использовать для нейтральных к рынку стратегий с минимальным риском.
Скринеры
📈 ● АКТИВЕН
ФЬЮЧЕРСНЫЙ АРБИТРАЖ
Futures / Futures · CEX-CEX
Поиск расхождений цен между фьючерсными контрактами на разных биржах. Одновременный вход LONG на одной бирже и SHORT на другой нейтрализует рыночный риск.
8 бирж Live данные WS книга P&L калькулятор
🔄 СКОРО
СПОТ-ФЬЮЧЕРСНЫЙ
Spot / Futures · CEX-CEX
Арбитраж между спотовым рынком одной биржи и фьючерсным контрактом другой. Учитывает стоимость переноса позиции и ставку фандинга.
8 бирж Фандинг учёт
💰 СКОРО
ФАНДИНГ АРБИТРАЖ
Funding Rate · Yield
Мониторинг аномальных ставок фандинга. Поиск возможностей для сбора положительного фандинга при нейтральной позиции через хедж на споте.
8 бирж Ставки 3 периода
⛓ В ПЛАНЕ
DEX-CEX АРБИТРАЖ
Decentralized / Centralized
Расхождения цен между децентрализованными протоколами (Uniswap, dYdX) и централизованными биржами. Требует on-chain исполнения.
EVM Solana Gas учёт
🔀 В ПЛАНЕ
ТРЕУГОЛЬНЫЙ АРБИТРАЖ
Triangular · CEX Internal
Поиск неэффективности внутри одной биржи через цепочку из трёх пар. BTC → ETH → USDT → BTC с положительным итогом.
Внутрибиржевой Без переводов
📊 В ПЛАНЕ
СТАТИСТИЧЕСКИЙ
Statistical · Pairs Trading
Торговля коинтегрированными парами на основе исторических корреляций. BTC/ETH, BNB/SOL — возврат к среднему при отклонении.
Z-score Корреляция
О платформе
Как это работает
Агрегация данных — REST и WebSocket запросы к 8 биржам каждые 60 секунд
Расчёт спреда — реальные цены с учётом глубины книги ордеров (VWAP)
Фильтрация — по бирже, стратегии, спреду, фандингу, статусу переводов
Мониторинг позиции — P&L калькулятор и графики в реальном времени
Поддерживаемые биржи
Binance, Bybit, OKX — прямой API без прокси
Gate.io, BingX, MEXC, KuCoin — через CORS proxy
Bitget — прямой API
Другие биржи — в процессе добавления
ℹ️
AXIOM находится в активной разработке. На данный момент доступен только скринер фьючерсного арбитража (CEX-CEX). Остальные модули будут запускаться поочерёдно по мере готовности.
